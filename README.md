# Cosmic Insights - Astrology Web App

A beautiful, simple, and responsive astrology web application built with vanilla HTML, CSS, and JavaScript.

## Features

- **Zodiac Explorer**: View personality details, elements, and ruling planets for all 12 signs.
- **Daily Horoscopes**: Fetch real-time readings for "Today" and "Tomorrow" using multiple fallback APIs for reliability.
- **Love Calculator**: Calculate compatibility scores across 5 categories with beautiful visualizations.
- **Responsive Design**: Works perfectly on mobile and desktop.

## How to use

Simply open `index.html` in any web browser to start the application.

## Hosting on GitHub Pages

1. Upload these files (`index.html`, `style.css`, `script.js`) to a GitHub repository.
2. Go to **Settings > Pages**.
3. Select the branch (usually `main`) and click **Save**.
4. Your site will be live at `https://yourusername.github.io/your-repo-name/`.
